# 🐟 Fish Farming Guide App  

> 📱 A real-world mobile app for managing fish farms with authentication, data tracking, and API-based operations.

---

## 🛠️ Tech Stack  
- React Native  
- Node.js (Express)  
- MySQL  

---

## 📱 Features  
- 🔐 Login / Signup Authentication  
- 🔄 Full CRUD Operations  
- 🌐 REST API Integration  
- 📊 Data Management  
- 🎨 Clean UI
